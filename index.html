About Me

Full-stack Software Developer!

Hello! I'm Cornelius Ngure, a Full-Stack Software Developer with a strong foundation in the MERN stack, Python, and Flask. I integrate technical proficiency with business strategy (from my BSc in Entrepreneurship knowledge), to build scalable, user-centric applications that create real impact.

My core expertise lies in developing RESTful APIs, designing robust databases using MySQL, PostgreSQL, and MongoDB, and implementing DevOps practices such as Docker and CI/CD pipelines to ensure efficient and reliable deployments.

I'm passionate about continuous learning, currently expanding into advanced DevOps and cloud engineering, and I enjoy building products that solve meaningful problems and drive business growth.

My Services

Web Development (Front-end & Back-end)

  • Front-End Development: Responsive websites using React.js, HTML, CSS, JavaScript
  • Back-End Development: RESTful APIs and server logic using Node.js, Express.js, Python, Flask
  • Full-Stack Solutions: End-to-end application development integrating frontend, backend, and databases

Database Design & Management

  • Database Design: Schema design for MySQL, PostgreSQL, MongoDB
  • Database Integration: Connecting and optimising databases within web applications
  • Data Migration & Optimisation: Improving performance, migrations, and queries

DevOps & Deployment

  • Containerisation: Dockerising applications for consistency across environments
  • CI/CD Pipelines: Automating build, test, and deployment workflows (GitHub Actions)

API Development & Integration

  • Custom API Development: Building secure and scalable APIs for various use cases
  • Third-Party API Integration: Integrating services like payment gateways, external data sources, or transcription APIs (e.g. AssemblyAI as in SoundScript)

Technical Consulting & Problem Solving

  • Technical Strategy Consulting: Advising individuals or small businesses on technology choices
  • Code Reviews & Debugging: Reviewing existing codebases to improve quality, readability, and efficiency
